# Remnawave Telegram Shop Bot Documentation

This repository contains the Mintlify documentation site for Remnawave Telegram Shop Bot.

## Development

Install the Mintlify CLI to preview your documentation changes locally:

```
npm i -g mint
```

Run the following command at the root of the documentation (where `docs.json` is located):

```
mint dev
```

View your local preview at `http://localhost:3000`.
