Приватная версия включает полнофункциональную **UI админ-панель** и **Личный кабинет пользователя**, построенные как Telegram Mini App. Ниже представлен обзор всех доступных функций.

## Личный кабинет пользователя (Mini App)

Персональный аккаунт пользователя со следующими функциями:

- **Информация о подписке** - текущий тариф, дата окончания, использование трафика
- **Доступные тарифы** - просмотр и покупка подписок
- **История платежей** - просмотр всех покупок
- **Промокоды** - применение промокодов для скидок
- **Активация триала** - активация пробной подписки
- **Управление устройствами** - просмотр и удаление устройств (HWID)
- **Настройки автоплатежа** - включение/выключение автоматического продления
- **Способы оплаты** - управление сохранёнными способами оплаты
- **Смена тарифа** - повышение/понижение подписки
- **Реферальная система** - реферальная ссылка и бонусы

### Скриншоты

#### Главная страница
Статус подписки, использование трафика, количество устройств, опции оплаты, автопродление и сохранённые карты.

![Главная страница](/ru/images/lk/SCR-20251221-sgok.png)

## Роли и разрешения

Полная система RBAC (ролевой контроль доступа):

- Создание/Обновление/Удаление ролей
- Назначение разрешений ролям
- Просмотр разрешений пользователя

### Доступные разрешения
| Разрешение | Описание |
|------------|----------|
| `view_dashboard` | Просмотр статистики дашборда |
| `view_clients` | Просмотр информации о пользователях |
| `edit_clients` | Изменение данных пользователей |
| `view_broadcasts` | Просмотр рассылок |
| `create_broadcast` | Создание/удаление рассылок |
| `view_promos` | Просмотр промокодов |
| `manage_promos` | Управление промокодами |
| `view_support` | Просмотр тикетов поддержки |
| `reply_support` | Ответы и управление тикетами |
| `view_partners` | Просмотр информации о партнёрах |
| `manage_partners` | Управление партнёрами |
| `view_settings` | Просмотр настроек |
| `manage_settings` | Изменение настроек |
| `view_plans` | Просмотр тарифов |
| `manage_plans` | Управление тарифами |
