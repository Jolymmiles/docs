## Что такое сквады?

Сквады - это группы в Remnawave. Вы можете выборочно назначать, к каким сквадам получают доступ пользователи.

## Конфигурация внутренних сквадов

```bash
SQUAD_UUIDS=773db654-a8b2-413a-a50b-75c3536238fd,bc979bdd-f1fa-4d94-8a51-38a0f518a2a2
```

- Настройте UUIDs сквадов, разделённые запятой
- Только соответствующие сквады назначаются пользователям
- Если пусто, назначаются все доступные сквады

## Конфигурация внешних сквадов

```bash
EXTERNAL_SQUAD_UUID=773db654-a8b2-413a-a50b-75c3536238fd
```

- Один UUID внешнего сквада
- Включается во все запросы создания/обновления пользователя
- Оставьте пусто для отключения

## Конфигурация сквадов для пробных периодов

Отдельные сквады можно назначить пробным пользователям:

```bash
TRIAL_INTERNAL_SQUADS=773db654-a8b2-413a-a50b-75c3536238fd,bc979bdd-f1fa-4d94-8a51-38a0f518a2a2
TRIAL_EXTERNAL_SQUAD_UUID=773db654-a8b2-413a-a50b-75c3536238fd
```

### Поведение

- **Если установлено**: Пробные пользователи назначаются на пробные сквады
- **Если не установлено**: Пробные пользователи используют обычную конфигурацию сквадов (резервное поведение)
- **Вариант использования**: Изоляция пробного трафика для предотвращения истощения ресурсов

## Пример конфигурации


```bash
# Обычные пользователи получают доступ к нескольким сквадам
SQUAD_UUIDS=773db654-a8b2-413a-a50b-75c3536238fd,bc979bdd-f1fa-4d94-8a51-38a0f518a2a2
EXTERNAL_SQUAD_UUID=773db654-a8b2-413a-a50b-75c3536238fd

# Пробные пользователи получают отдельные пробные сквады
TRIAL_INTERNAL_SQUADS=abc12345-def6-7890-ghij-klmnopqrstuv
TRIAL_EXTERNAL_SQUAD_UUID=xyz98765-abc4-3210-defg-hijklmnopqrs
```
