## Управление подписками

### Несколько планов

- **1 месяц**: Быстрый пробный или краткосрочный доступ
- **3 месяца**: Популярный среднесрочный вариант
- **6 месяцев**: Лучшее соотношение цены и качества
- **12 месяцев**: Лучшее значение с наибольшей скидкой

### Автоматические уведомления

- Запускается ежедневно в **16:00 UTC**
- Уведомляет пользователей **за 3 дня** до истечения
- Включает точную дату истечения
- Предоставляет удобную кнопку продления
- Поддержка нескольких языков (русский/английский)

## Управление пользователями

### Встроенная фильтрация

- **Обнаружение подозрительных аккаунтов**: Предотвращает фишинг аккаунты
- **Поддержка белого списка**: Обход проверок для доверенных пользователей
- **Поддержка чёрного списка**: Блокировка конкретных пользователей
- **Распознавание служебных аккаунтов**: Разрешить легитимные боты

### Система пробных периодов

```bash
TRIAL_DAYS=7
TRIAL_TRAFFIC_LIMIT=10
TRIAL_REMNAWAVE_TAG=trial_user
```

- Привлекайте новых пользователей бесплатной пробой
- Ограничьте трафик для предотвращения злоупотреблений
- Отдельно отслеживайте пробных пользователей
- Опциональное отдельное назначение сквадов

## Динамический интерфейс

Кнопки показываются/скрываются в зависимости от конфигурации:

| Кнопка | Обязательная переменная | Вариант использования |
|--------|------------------------|----------------------|
| Главное меню | Всегда показана | Навигация |
| Купить | Всегда показана | Покупка подписки |
| Подключить | Всегда показана | Получить конфиг VPN |
| Статус сервера | `SERVER_STATUS_URL` | Здоровье системы |
| Поддержка | `SUPPORT_URL` | Получить помощь |
| Обратная связь | `FEEDBACK_URL` | Отправить отзыв |
| Канал | `CHANNEL_URL` | Сообщество |
| Условия использования | `TOS_URL` | Юридическая информация |

## Многоязычная поддержка

Поддерживаемые языки:
- **Русский** (по умолчанию)
- **Английский**

```bash
DEFAULT_LANGUAGE=ru
```

Все сообщения настраиваются через файлы переводов в папке `translations/`.

## Проверки здоровья

Бот предоставляет конечные точки мониторинга:

```bash
curl http://your-bot:8080/healthcheck
```

Проверяет:
- Подключение к БД
- Доступность панели Remnawave
- Статус службы бота
- Действительность API ключа

## Команды администратора

```
/sync - Синхронизировать пользователей из Remnawave
```

Загружает всех пользователей с панели и обновляет БД, удаляя тех, кого больше нет в Remnawave.

## Управление сквадами

### Внутренние сквады
- Управление методами подключения, к которым получают доступ пользователи
- Несколько сквадов на пользователя
- Независимая конфигурация для пробных

### Внешние сквады
- Один UUID сквада
- Включается во все операции с пользователем
- Опциональное назначение

## Управление трафиком

```bash
TRAFFIC_LIMIT=100
TRIAL_TRAFFIC_LIMIT=10
```

- Установите лимит трафика на подписку
- Отдельные лимиты для обычных и пробных пользователей
- 0 = неограниченный трафик

## Система рекомендаций (опциональная)

```bash
REFERRAL_DAYS=30
```

- Отслеживание рекомендаций пользователями
- Отключено, если установлено значение 0
