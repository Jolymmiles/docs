Бот поддерживает четыре гибких способа оплаты для удовлетворения различных предпочтений пользователей.

## YooKassa (Банковские карты)

💳 Принимайте Visa, Mastercard и другие платёжные карты

**Лучше всего для**: Регион СНГ, платежи по карте
**Ограничения**: Географические ограничения

## CryptoPay (Криптовалюта)

🪙 Bitcoin, Ethereum и другие криптовалюты

**Лучше всего для**: Глобальная аудитория, технически подкованные пользователи
**Преимущества**: Нет чарджбэков, мгновенное поселение доступно

## Telegram Stars

⭐ Встроенная платёжная система Telegram

**Лучше всего для**: Безупречная интеграция Telegram
**Преимущества**: Пользователи знакомы с платформой, глобальная доступность
**Ограничения**: Ограниченное преобразование, только целые цены

## Tribute (Подписки)

🎟️ Управление подписками с автоматическим продлением

**Лучше всего для**: Автоматические продления, корпоративная установка
**Требования**: Общедоступный домен с SSL сертификатом

## Конфигурация платежа

Каждый способ оплаты настраивается независимо:

```bash
# Включение/отключение способов
YOOKASA_ENABLED=true
CRYPTO_PAY_ENABLED=true
TELEGRAM_STARS_ENABLED=true

# Установите цены для каждого способа
PRICE_1=5
PRICE_3=12
STARS_PRICE_1=50
```

См. подробные руководства настройки для каждого способа оплаты в разделе **Платежные системы**.
