## Payment Configuration

Each payment method configured independently:

```bash
# Enable/disable methods
YOOKASA_ENABLED=true
CRYPTO_PAY_ENABLED=true
TELEGRAM_STARS_ENABLED=true

# Set prices for each method
PRICE_1=5
PRICE_3=12
STARS_PRICE_1=50
```
