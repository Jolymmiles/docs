The private version includes a full-featured **Admin UI Panel** and **User Dashboard** built as a Telegram Mini App. Below is an overview of all available features.

## User Dashboard (Mini App)

Personal account for users with the following features:

- **Subscription info** - view current plan, expiration date, traffic usage
- **Available plans** - browse and purchase subscription plans
- **Payment history** - view all purchases
- **Promo codes** - apply promo codes for discounts
- **Trial activation** - activate trial subscription
- **Device management** - view and delete devices (HWID)
- **Autopay settings** - enable/disable automatic renewal
- **Payment methods** - manage saved payment methods
- **Plan change** - upgrade/downgrade subscription
- **Referral system** - referral link and bonuses
